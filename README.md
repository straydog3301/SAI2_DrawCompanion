# SAI2 DrawCompanion 🎨 (v1.3.0)

`SAI2 DrawCompanion` (繪圖助手) 是一款專門為 **PaintTool SAI Ver.2** 開發的開源輔助工具。它能自動在背景追蹤您的繪圖時間、提供多維度統計圖表，並具備智慧型縮時錄影功能與液化變形工具。

---

## 🌟 主要功能 (Core Features)

### 1. 🕒 高精度時間追蹤 (Draw Time Tracker)
- 自動偵測並累計 SAI2 中各專案/圖檔的「開啟時間」與實際的「動筆作畫時間」（支援繪圖板筆壓與滑鼠按壓偵測）。
- 當您暫停作畫超過設定的閒置時間（例如 10 秒）時，計時會自動暫停，並在您下筆時瞬間恢復，確保數據 100% 精準。
- 依照檔案所在的第一層資料夾**自動群組分類**，亦支援自訂分組。

### 2. 📈 視覺化統計圖表 (Visual Statistics)
- 內建統計儀表板，支援切換查看：週 / 月 / 三個月 / 六月 / 年 / 至今。
- 視覺化長條圖展示每日總作畫時長、平均作畫時間與單日作畫高峰。

### 3. 🎬 智慧縮時錄影 (Smart Timelapse)
- **僅錄製畫布**：高精度裁切畫布範圍（分頁模式與浮動視窗模式皆完美支援），自動過濾上方選單、調色盤及彈出式對話框，不干擾畫面。
- **錄製 SAI 視窗** / **整個螢幕**：提供多種錄製範圍供選擇。
- **動態播放倍數**：支援 1x 到 100x 的作畫加速。

### 4. 🎞️ 四種影片品質規格 (4 Video Quality Presets)
- **輕量空間 (Compact)**：以 JPEG 75% 壓縮畫格，高壓縮率。
- **標準畫質 (Standard)**：以 JPEG 90% 壓縮，畫質與空間的完美平衡（預設）。
- **專業高清 (High-Quality)**：以 JPEG 98% 壓縮，保留極致細節。
- **100% 原畫無損 (Lossless)**：以 PNG 無損格式保存畫格，搭配 FFmpeg CRF 0 合成無損 MP4。

### 5. 📌 最近使用優先 (Recently Active Sort)
- 列表預設會將「最近正在繪製的專案分組與檔案」優先挪到列表第一順位，方便密集開啟編輯。
- 樹狀列表支援點擊標頭欄位（開啟時間、動筆時間、最後使用等）即時進行雙向排序。

### 6. 🗺️ 多語系支援 (Dynamic i18n Support)
- 軟體開箱即內建：**繁體中文 (Traditional Chinese)**、**英文 (English)** 與 **日文 (Japanese)**。
- 支援外部自訂語系：只需將翻譯好的 JSON 語系檔案丟入與 `.exe` 同一層的 `locales/` 資料夾，即可直接在設定選單內切換，**完全不需要重新編譯專案**。
- 當語系檔案有缺漏字詞時，會自動套用英文或中文的對應欄位，保證介面安全回退。

### 7. 🎨 高級主題與 Windows 標題列整合 (Modern Styling)
- 支援深色模式 (Dark Mode) 與亮色模式 (Light Mode) 快速切換。
- 提供 12 種精緻的預設主題色（羅蘭紫、向日葵黃、天空藍等）與自訂 Hex 色碼。
- **標題列整合**：軟體標題列（TitleBar）會自動套用主題模式與主色調。

### 8. 📱 極簡懸浮視窗與快捷鍵 (Mini Widget & Hotkey)
- 提供可任意拖曳、置頂的迷你小標籤，展示目前圖檔的作畫進度。
- 支援全域錄影開關快速鍵（預設為 F10），方便在繪圖時快速暫停或開啟錄製。

### 9. 🌀 液化變形工具 (Liquify / Warp Tool)
SAI2 本身不內建液化功能，本工具提供了一套完整的液化變形方案，讓您無需離開 SAI2 工作流程即可進行變形操作。

**使用方式**：在 SAI2 中先選取要液化的區域，然後按下 `Ctrl + Alt + L` 即可開啟液化編輯器。

**已實現功能**：
- ✅ **推拉變形 (Push)**：用筆刷拖曳即可推動像素，直覺操作
- ✅ **膨脹 (Bloat)** / **收縮 (Pinch)**：按住即可讓區域膨脹或收縮
- ✅ **順時針旋轉 (CW Twirl)** / **逆時針旋轉 (CCW Twirl)**：讓筆刷範圍內的像素旋轉扭曲
- ✅ **重置筆刷 (Reset Brush)**：將筆刷範圍內的變形恢復原狀
- ✅ **筆刷大小 / 強度調整**：滑桿調整或快捷鍵 `[` / `]` 調整筆刷大小
- ✅ **Undo / Redo**：支援最多 50 步歷史紀錄，`Ctrl+Z` 復原、`Ctrl+Y` 重做
- ✅ **縮放與平移**：`Ctrl + 滾輪` 縮放、`空白鍵 + 拖曳` 平移畫布
- ✅ **背景模式切換**：棋盤方格 / 純白背景 / 灰色背景，設定自動記憶
- ✅ **自動複製並貼回 SAI2**：套用後自動複製到剪貼簿（含透明度），並模擬 `Ctrl+V` 貼回 SAI2
- ✅ **快捷鍵操作**：`Enter` 套用並關閉、`Esc` 取消
- ✅ **筆刷設定持久化**：筆刷大小、強度、背景模式自動保存，下次開啟時還原

**尚未實現 / 已知限制**：
- ⬜ **原地貼回對齊 (Paste-in-Place)**：目前液化完成後，圖像會被貼到畫布左上角，需要使用者手動對齊。自動座標比對機制已部分實現（畫布截取與像素匹配），但因 SAI2 剪貼簿行為的限制，尚無法可靠地取得完整畫布以計算選區偏移量。
- ⬜ **平滑筆刷 (Smooth)**：液化筆刷模式中的「平滑」功能（將變形過渡做得更自然平滑）尚未實現。
- ⬜ **凍結遮罩 (Freeze Mask)**：允許使用者先塗抹遮罩區域使其不受液化影響，防止變形擴散到不想動的部分。
- ⬜ **液化網格顯示 (Show Mesh)**：在編輯時可選擇顯示底層的液化網格，便於觀察變形程度。
- ⬜ **壓感支援 (Pen Pressure)**：根據繪圖板的筆壓動態調整液化強度。

## 🛠️ 開發與編譯方式 (Development & Building)

### 準備工作 (Prerequisites)
1. **Python 3.10+** (確保已勾選加入系統 Path)。
2. **FFmpeg**：由於錄影合成影片需要使用 FFmpeg，請確保本機已安裝且其 `bin` 目錄已被加入到系統的 `PATH` 環境變數中。

### 安裝依賴 (Install Dependencies)
在專案目錄下執行以下指令安裝需要的 Python 套件：
```bash
pip install -r requirements.txt
```

### 打包為獨立執行檔 (Compile to Standalone EXE)
執行 `build.bat`，程式將自動安裝 PyInstaller 並將主程式編譯為單一免安裝的執行檔。編譯完成後，您可以在 `dist/` 資料夾下找到 `SAI2_DrawCompanion.exe` 與對應的 `locales` 語系目錄。
```bash
build.bat
```

## 🚀 使用說明 (Usage)

1. 開啟 `SAI2_DrawCompanion.exe`。
2. 啟動您的 **PaintTool SAI Ver.2**。
3. 開始作畫，程式會自動捕捉目前工作的專案名稱，並開始累積作畫時間與錄製縮時畫格。
4. **自動開機啟動**：您可以執行 `setup_startup.bat`，這將會在系統的啟動目錄中建立捷徑。當 SAI2 開啟時，本軟體將會自動在背景啟動；當 SAI2 關閉時，本軟體也會自動儲存並關閉。若想移除捷徑，只需執行 `remove_startup.bat`。

### 液化工具快捷鍵一覽

| 快捷鍵 | 功能 |
|---|---|
| `Ctrl + Alt + L` | 開啟液化編輯器（需先在 SAI2 中建立選區） |
| `Enter` | 套用液化結果並貼回 SAI2 |
| `Esc` | 取消液化並關閉編輯器 |
| `[` / `]` | 縮小 / 放大筆刷尺寸 |
| `Ctrl + Z` | 復原 (Undo) |
| `Ctrl + Y` | 重做 (Redo) |
| `Ctrl + 滾輪` | 縮放畫布 |
| `空白鍵 + 拖曳` | 平移畫布 |
